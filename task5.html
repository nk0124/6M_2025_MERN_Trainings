CCA -Certificate Course in Computer Application

What is CCA certificate?

CCA,an acronym for Certificate in Computer Application ,is a vocational certification program that aims to upgrade an individual's computer skills , enhancing their employability.

What is the qualification for CCA?

Candidates must have passed graduation or equivalent level (with Mathematics as a subject of study at 10+2 or graduation level) examination from a recoginzed university . He/she must have obtained at least 50% (relaxable for reserved category candidates ) of the marks at graduattion or equivalent level.

Which subjects is CCA?

Certificate in Computer Application is a certificate course to study computer Application in depth.The course inparts theoretical , practical and technical knoweledge to its learns about various computer tools that are used in day to day life.

C++ - Object Oriented Programming using C++

What is C++?

What Does plus plus Programming Language Mean?C++ (said C plus plus) is an object-oriented computer language created by notable computer scientist Bjorne Stroustrop as part of the evolution of the C family of language .

what is Different between C and C++?

C is a structural or procedural programming language that was for system applications and low - level programming applications.Whereas C++ is an object oriented programming language having some additional features like Encapsulation. Data Hiding, Data Abstraction.Inheritance , Polymorphism , etc.

What is the Oops Concept

Object-oriented programming (OOP) is a computer programming model that organizes software designed around data,objects rather than function and logic .An object can be defined as a data field that has unique attributes and behavior.
